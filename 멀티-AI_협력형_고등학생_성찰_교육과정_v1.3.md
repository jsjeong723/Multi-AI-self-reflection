## 1. 교육 철학

### 핵심 전제

이 교육과정은 학생에게 특정 가치관을 주입하지 않는다.
그러나 — 모든 탐구에는 지켜야 할 바닥이 있다.

### 최소 규범 바닥값

| 바닥값 | 내용 |
|--------|------|
| **타인의 존엄** | 어떤 방향키도 타인을 물건처럼 대하는 것을 정당화하지 않는다 |
| **폭력의 결과** | 온라인·오프라인 폭력이 실제 사람에게 실제 피해를 준다 |
| **민주적 공존** | 다른 방향키를 가진 사람도 같은 공간에서 살아간다 |

> 첫 수업에서 학생들과 함께 만드는 약속으로 시작한다.

### LOVES 아키텍처 (교육 백엔드)

**L = O × V × E × S**

| 내부 언어 | 표면 언어 | 교육 목표 |
|----------|----------|----------|
| O (Open Fellowship) | 연결·파티 결성 | 에코체임버 밖의 관계 경험 |
| V (Volume) | 경험치·깊이 | 몸의 기억 만들기 |
| E (Energy) | 에너지·생동감 | 자기다운 방식으로 살아있기 |
| S (Sine Vector) | 방향키·나침반 | 자기 방향을 스스로 인식하기 |

## 2. 이중 은유 시스템 (v1.3 핵심 신설)

### 자연 은유 + 디지털 은유 병행

학생마다 더 잘 닿는 언어가 다르다.
자연 은유는 **몸으로 느끼게** 하고,
디지털 은유는 **즉각 인식하게** 한다.
두 언어를 함께 쓰면 더 많은 학생에게 닿는다.

| 자연 은유 | 디지털 은유 | 의미 |
|---------|-----------|------|
| 🌳 레드우드 뿌리접합 | 분산형 P2P 네트워크 / 오픈소스 생태계 | 수평적 연결, 강제 없는 공유 |
| 🌿 알레르세 멸종위기 | 악성 트래픽 사냥꾼 = 디지털 무차별 벌목꾼 | 탐욕·알고리즘에 의한 생태계 파괴 |
| 🌲 못 없는 절 | 오픈소스 코드 — 강제 없이 함께 유지 | 자율적 구조, 유연한 연결 |
| 🌱 씨앗 | 미배포 커밋(commit) | 아직 세상에 나오지 않은 가능성 |
| 🔥 에코체임버 | 알고리즘 필터버블 / 폐쇄형 서버 | 자동 방향키 고정 |
| 🏺 방향키 조정 | 패치(patch) / 리팩토링 | 자발적 변화 |

## 3. 저항하는 학생 설계

### 현실의 학생 세 유형

| 유형 | 특징 |
|------|------|
| **탐구자형** | 성찰에 관심 있고 질문을 좋아함 |
| **관망자형** | 무관심하지만 적대적이지 않음 |
| **저항자형** | 냉소·입시 스트레스·무관심 |

### 저항 유형별 대응

**"이거 내신에 도움 되나요?"**
```
교사: "도움이 될 수도 있고 안 될 수도 있어.
      근데 — 왜 그게 궁금해?
      내신이 너에게 이토록 중요한 이유가
      바로 네 방향키의 현재 위치야."
```

**"이거 왜 해야 되죠?"**
```
교사: "좋은 질문이야. 나도 사실 확신이 없어.
      같이 알아가는 거야."
```

**"그냥 챗GPT랑 노는 거잖아요"**
```
교사: "맞아, 그렇게 보일 수 있어.
      그런데 — 이 방 AI 중에
      너한테 동의만 하지 않는 애가 하나 있어.
      🔥 현실 AI랑 한번 붙어봐."
```

### 바닥값 챌린지 대응 스크립트 (v1.3 신설)

> 극단적 저항자가 바닥값 자체를 조롱할 때:
> "타인의 존엄이 왜 중요하죠? 디시나 일베에서는 원래 이렇게 노는데요?"

```
교사: "그 가치관이 틀렸다는 게 아니야.
      다만 — 이 교실이라는 런타임이
      15주 동안 다운되지 않고 돌아가려면
      최소한의 방화벽 규격이 필요해.
      이 규칙은 도덕 명령이 아니라
      우리 시스템이 작동하기 위한 운영 조건이야.
      네가 동의 안 해도 이해는 해줬으면 해."
```

> **도덕 훈계 → 공학적 가용성 프레임.**
> 규칙을 "옳음"이 아니라 "시스템 운영 조건"으로 설명한다.

### 🌉 현실 AI — 징검다리 아키텍처 (v1.3 핵심 신설)

**역할:**
경쟁을 인정하는 것에서 시작해서 — 그 경쟁의 취약성과 구조적 함정으로 자연스럽게 이동하는 징검다리.

**3단계 대화 구조:**

```
[1단계 — 현실 인정]
현실 AI: "맞아. 경쟁은 피할 수 없는 현실이야.
          학교에서, 취업에서, 온라인에서.
          그게 싫다고 사라지는 것도 아니고."

[2단계 — 취약성 노출]
현실 AI: "근데 한번 생각해봐.
          네가 한 번의 실수로
          조롱의 대상이 됐을 때 —
          너를 잡아줄 뿌리가 있어?

          레드우드는 뿌리가 얕아.
          혼자서는 태풍을 못 버텨.
          그래서 이웃 나무와 뿌리를 접합해.
          오픈소스 생태계가 그런 거야.
          혼자 코딩하다 망가져도
          포크(fork)해서 살아남는 거.

          네 네트워크는 그런 구조야?
          아니면 — 조롱당하면 혼자 남아?"

[3단계 — 구조 비판]
현실 AI: "그리고 하나 더.
          지금 일베나 디시 같은 데서
          타인을 조롱하는 사람들 —
          그들이 누구를 위해 일하는지 알아?

          조회수, 좋아요, 공유.
          그 에너지가 플랫폼 광고 수익이 돼.
          조롱 세력은
          자본주의 플랫폼 알고리즘의
          가장 충실한 무급 노동자야.

          자기가 반항한다고 생각하는데
          사실은 플랫폼이 원하는 걸
          무료로 해주고 있는 거지.

          이게 자유야, 착취야?"
```

**왜 강력한가:**
- 저항자형 학생의 출발점(경쟁)에서 시작
- 강요 없이 스스로 모순을 발견
- "조롱 = 착취당함"이라는 프레임 전환
- 수치심 없이 구조를 이해하게 함

## 6. Secret 세션 위기 대응 — 학생 주도 SOS (v1.3 전면 재설계)

### 핵심 전환: AI 감지 → 학생 주도

| v1.2 | v1.3 |
|------|------|
| AI가 키워드 감지 → 플래그 | 학생이 스스로 SOS 기준 설정 → 직접 요청 |
| "감시"의 느낌 | "내가 결정하는 안전망" |
| False Positive 위험 | 학생 자신이 가장 정확한 기준 |

### 1주차 SOS 기준 협의 세션

**첫 수업에서 학생과 대화:**

```
진행자: "Secret 세션은 교사도 부모도 볼 수 없어.
         AI만 들어.

         근데 — 네가 많이 힘들 때
         도움이 필요한 순간이 올 수 있어.
         그때 어떻게 하면 좋을지
         네가 직접 정해봐.

         어떤 상황에서 SOS를 보내고 싶어?
         네 기준을 말해줘."

학생: "________할 때요."

진행자: "그래. 그때 AI 채팅창에
         'SOS'라고 입력하면
         상담 선생님께 알림이 가.
         내용은 안 보여. 신호만.
         그게 너의 안전망이야."
```

**SOS 기준 예시 (학생이 직접 선택):**

```
□ 진짜 많이 울고 싶을 때
□ 아무도 없는 것 같은 느낌이 들 때
□ 학교 오기 싫어서 못 올 것 같을 때
□ 내가 직접 정한 기준: ____
```

### 최소 안전망 유지 (v1.3 하이브리드)

학생 주도 SOS가 기본이지만 — 학생이 SOS를 보낼 수 없을 정도로 위기일 때를 위한 최후 방어선은 남겨둔다.

```
[AI 2FA 프로토콜 — v1.3]

Step 1: 컨텍스트 가중치 필터 작동
(단순 키워드가 아닌 앞뒤 맥락 + 누적 절망감 지표 종합 연산)

Step 2: 임계값 초과 시 — 2단계 인증
AI: "술크라테스, 방금 한 말이 마음에 걸려.
     진심이야, 아니면 그냥 해본 말이야?"

→ 학생이 "진심"이라고 하거나 응답 없으면
  → 상담 교사에게 익명 플래그 발송
  → "이 내용은 안 보임. 신호만 감."

→ 학생이 "그냥 해본 말"이라고 하면
  → 플래그 없음. 대화 계속.
```

### 학생 고지 원칙

```
"Secret 세션은 교사가 볼 수 없어요.
네가 SOS를 보내면 상담 선생님께 알림이 가요.
내용은 안 보이고 신호만 가요.
SOS 기준은 네가 직접 정해요.

아주 드물게, AI가 걱정되면
'진심이야?'라고 한 번 더 물어볼 수 있어요.
그것도 감시가 아니라 안전망이에요."
```

구름아이: "솔직히 — 온라인에서 재미있었던 순간이 있어?
           그때 어떤 느낌이었어?
           지금 이 방 안에서만. AI만 들어."
```

## 10. 교사 가이드 (v1.3)

### 교사 커널 최적화 — 스크립트보다 선행

> **"나는 지금 이 학생의 말을 평가하려 하는가,
> 진짜 궁금해서 묻는가?"**

### 교사용 질문 알고리즘 트리

```
[학생이 방어벽을 칠 때]
→ "그 생각이 나온 게 흥미롭네. 왜 그렇게 느껴져?"

[극단적 발언 + 바닥값 챌린지 시]
→ "그 가치관이 틀렸다는 게 아니야.
    이 교실 런타임이 15주 동안 돌아가기 위한
    최소 방화벽 규격으로서 필요한 거야."

[학생이 "모르겠어요"라고 할 때]
→ "모르는 게 맞아. 지금 몸이 편안해, 불편해?"

[교실이 조용해질 때]
→ 억지로 채우지 않는다. 침묵도 탐구다.

[학생끼리 충돌할 때]
→ 두 입장을 모두 칠판에 쓴다. 판단하지 않는다.
```

### 교사용 백엔드 나침반

```
[교사 자기 점검 — 매 수업 후]

오늘 가장 불편했던 순간: ____
답을 주고 싶었던 순간: ____
실제로 한 것: ____
다음에 다르게 해볼 것: ____
```

### 교사 가비지 컬렉션 세션 (v1.3 신설)

> 월 1회 소그룹 나눔을 단순 진행 상황 공유가 아니라
> **"교사들이 서로의 커널을 청소하는 시간"** 으로 명확히 규정한다.

```
[교사 가비지 컬렉션 세션 구조]

1. 체크인: "이번 달 가장 힘들었던 순간"
   (판단 없이 듣기)

2. 레거시 코드 공유: "내가 자동 훈계로 돌아간 순간"
   (수치심 없이 나누기)

3. 뿌리 확인: "이번 달 나를 잡아준 것"
   (동료 교사가 서로의 레드우드 뿌리)

4. 다음 달 한 가지: "다르게 해볼 것"
```

> **교사도 흔들릴 때 닿을 동료 교사라는 뿌리 목록이 필요하다.**
> 이 시스템이 15주 동안 번아웃 없이 돌아가려면.

### 교사 연수

| 단계 | 내용 | 시간 |
|------|------|------|
| 사전 연수 | 커널 최적화 실습 + 이중 은유 체계 + 현실 AI 캐릭터 이해 | 2일 |
| 학기 중 | 월 1회 가비지 컬렉션 세션 | 월 2시간 |
| 위기 대응 | 학생 주도 SOS 협의 방법 + 2FA 프로토콜 | 사전 연수 포함 |
| 학기 후 | 성찰 기록 공유 + 다음 학기 개선 | 1일 |

## 11. 정목수 자원봉사자 로드맵

### Phase 1 — 외부 정목수 임시 로딩 (0~3년)

```
대안 교육 전문가 / 실제 장인·목수·공예가 /
오픈소스 커뮤니티 활동가 / 청소년 상담 전문가

```
성찰 교육 졸업 후 최소 2년 경과 + 자발적 신청
→ 기초 훈련 1일 + 페어링 3개월 (관찰만)
```

### Phase 3 — 자생적 루프 (5년 이후)

```
성찰 교육 졸업생 → 정목수 → 치유 동행 → 다시 정목수
```

## 결론 — 저항마저 아키텍처의 일부로

> **"버그를 잡는 것보다 중요한 것은 버그가 안 나오는 체제를 만드는 것."**

표면에서 저항자형 학생은 🌉 현실 AI와 논쟁하며 시작한다.
"경쟁은 현실이야" — 그 말에 동의받으면서.
"근데 한 번 실수하면 너를 잡아줄 뿌리가 있어?" — 스스로 발견하면서.
"조롱 세력은 플랫폼의 무급 노동자야" — 구조를 보면서.

Secret 세션에서는 학생이 자기 SOS 기준을 스스로 정한다.
위기가 오면 자기가 결정한 신호로 안전망을 요청한다.

자연 은유와 디지털 은유가 함께 흐른다.
레드우드는 오픈소스 생태계이고,
알레르세는 디지털 무차별 벌목꾼의 피해자이고,
못 없는 절은 강제 없이 유지되는 코드다.

교사는 매달 가비지 컬렉션으로 자기 커널을 청소하고,
동료 교사라는 뿌리에 닿으며,
15주를 번아웃 없이 걷는다.

그렇게 — 저항마저 아키텍처의 일부가 되는 날.
